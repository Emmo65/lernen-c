# Übung 008 – Einfache Struktur

**Aufgabe:**  
Erstelle eine Struktur Auto mit den Attributen Marke (als String) und Geschwindigkeit (als Integer).  
Erstelle eine Variable dieser Struktur und weise Werte zu.  

**Hinweis:**  
Nutze struct Auto { ... }; und strcpy() aus <string.h> für Strings.  
C hat keine Klassen wie Java – stattdessen verwendet man struct (Strukturen) ohne Methoden oder Konstruktoren.

**Beispielausgabe:**
Marke: BMW
Geschwindigkeit: 180 km/h
