# Übung 005 – Einfache Funktion

**Aufgabe:**  
Erstelle eine Funktion, die eine Zahl als Parameter erhält und das Quadrat dieser Zahl zurückgibt.  
Hinweis: Verwende eturn, um das Ergebnis zurückzugeben. Deklariere die Funktion vor main().

**Beispielausgabe:**
Gebe eine Zahl ein um das Quadrat zu berechnen und bestaetige mit Enter: 5
Das Quadrat ist: 25
