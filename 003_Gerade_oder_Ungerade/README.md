# Übung 003 – Gerade oder ungerade?

**Aufgabe:**  
Erstelle ein Programm, das prüft, ob eine eingegebene Zahl gerade oder ungerade ist.  
Hinweis: Nutze den Modulo-Operator % und eine if-else-Verzweigung.

**Beispielausgabe:**
Bitte gib eine Zahl ein und bestätige mit Enter: 5
Deine Zahl 5 ist ungerade.
